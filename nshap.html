nshap.vfunc.interventional_shap(f, X, target=None, num_samples=1000, random_state=None, meomized=True)ÔÉÅ

Approximate the value function of interventional SHAP.

In order to compute n-Shapley Values, a data set is sampled once and then fixed for all evaluations of the value function.

Parameters:

  • f (function) ‚Äì The function to be explained. Will be called as f(x) where x has shape (1,d).

  • X (_type_) ‚Äì Sample from the data distribution.

  • target (int, optional) ‚Äì Target class. Required if the output of f(x) is multi-dimensional. Defaults to None.

  • num_samples (int, optional) ‚Äì The number of samples that should be drawn from X in order estimate the value function. Defaults to 1000.

  • random_state (_type_, optional) ‚Äì Random state that is passed to np.random.default_rng. Used for reproducibility. Defaults to None.

  • meomized (bool, optional) ‚Äì Whether the returned value function should be meomized. Defaults to True.

Returns:

The vaue function.

Return type:

function

class nshap.vfunc.memoized_vfunc(func)ÔÉÅ

Bases: object

Decorator to meomize a vfunc. Handles hashability of the numpy.ndarray and the list.

The hash depends only on the values of x that are in the subset of coordinates.

This function is able to handle parameters x of shape (d,) and (1,d).

Meomization helps to avoid duplicate evaluations of the value function, even if we perform computations that involve the same term multiple times.

https://wiki.python.org/moin/PythonDecoratorLibrary
